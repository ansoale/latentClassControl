# latentClassControl
This package is for implementing the latent class control method for correcting bias in 
observational studies.

- The results for Tables 1 and 2 and Figures 1-4 in the appendix can be obtained by 
running the code in Model 1-4.R

- Figure 2 can be obtained by running Bias_simulations.R

- Table 3 can be obtained by running Power_analysis.R

- Tables 4 and 5 and Figures 5 and 6 in the appendix can be obtained by running 
the Insurance_applications.R
